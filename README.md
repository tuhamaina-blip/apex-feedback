# Apex Advisory

A clean, responsive feedback platform built for **Apex Advisory** — a financial and strategic consulting firm dedicated to helping businesses grow, scale and make smarter decisions. Apex Advisory works with clients across a range of industries, offering expert guidance in areas such as business strategy and financial planning

This project was developed as a real-world client-style website with multiple pages, responsive layouts, and modern UI practices using TailwindCSS.

---

## Overview

Apex Advisory needed a professional online presence to showcase services, build trust, and allow clients to easily get in contact.

This website solves that by providing:
- A modern homepage with clear branding
- Dedicated services page
- Contact page for client inquiries
- Privacy policy for legal transparency
- Custom error page for better user experience

---

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with overview of services and CTA |
| About | `about.html` | Information about the company |
| Services | `services.html` | Breakdown of consulting services |
| Contact | `contact.html` | Contact form for client inquiries |
| Privacy Policy | `privacy.html` | Legal privacy information |
| Error Page | `error.html` | Custom error page for invalid routes |

---

## Built With

- **HTML5** — semantic structure and accessibility
- **Tailwind CSS** — utility-first styling and responsive design

---

## Features

- Fully responsive mobile-first design
- Multi-page navigation system
- Clean UI with consistent branding
- Contact form with validation
- Custom error page
- Footer with site-wide navigation links

---

## Responsive Design

The site uses Tailwind CSS breakpoints:

- **Mobile** — stacked layout for readability
- **Tablet** — adjusted spacing and grid layouts
- **Desktop** — multi-column layouts for better use of space
